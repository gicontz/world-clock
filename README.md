# Installation
Install dependencies of the project

## Yarn
`yarn`

## NPM
`npm install`

# Run
Execute Project in Dev mode

## Yarn
`yarn start`

## NPM
`npm run start`

# Build
Build Project to serve in production

## Yarn
`yarn build`

## NPM
`npm run build`
